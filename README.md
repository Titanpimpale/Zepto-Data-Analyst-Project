# Zepto-Data-Analyst-Project
# 🛒 Zepto Data Analysis Project

## 📌 Overview

Analyzed a retail dataset (Zepto products) using SQL to uncover insights related to pricing, discounts, and stock availability.

---

## 🎯 Objectives

* Explore product categories and data structure
* Clean invalid and inconsistent data
* Analyze pricing and discount patterns
* Identify stock and revenue insights

---

## 🧹 Data Cleaning

* Removed records with invalid pricing
* Standardized price formats for consistency

---

## 📊 Analysis & Insights

* Identified top discounted products
* Found high-value out-of-stock items (possible demand gaps)
* Analyzed revenue contribution by category
* Observed inconsistent discount strategies across categories

---

## 🛠️ Tools Used

* SQL (PostgreSQL)
* VS Code
